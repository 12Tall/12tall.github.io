`Hexo Next` 在使用中遇到的问题与结局办法汇总。  

<!--more-->


``` bash
$ hexo new "My New Post"  
$ hexo new -p  notation/pm2/pm2-简单教程
$ hexo clean  
$ hexo generate
$ hexo server  
$ hexo deploy
```
