[Vuepress2](https://v2.vuepress.vuejs.org/) 确实是一个强大的博客框架，但是对于我来说，它提供的功能还是有限。在尝试了[vuepress-theme-hope](https://vuepress-theme-hope.github.io/v2/) 这款主题后，感觉其基本能满足日常笔记的需要。于是决定将原博客的内容全部迁移至`vuepress-theme-hope`。
